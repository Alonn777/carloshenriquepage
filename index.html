Trabalhos Recentes!

JoiShop Eccomerce

Este é um protótipo de loja virtual inspirado no estilo de Eccomerces, desenvolvido utilizando HTML semântico, CSS avançado (com layouts responsivos utilizando Display Flex e Grid) e JavaScript para manipulação do DOM e requisição de dados de uma API do Mercado Livre. O projeto visa proporcionar uma experiência de navegação fluida e eficiente, com ênfase na responsividade e na interação dinâmica com o usuário.

Landig Page: Cafeteria

Este é um protótipo de cafeteria desenvolvido com HTML, CSS e JavaScript. O JavaScript é utilizado para manipulação do DOM, permitindo que, ao clicar no botão "Endereço", a localidade da cafeteria seja exibida dinamicamente. O projeto foca em proporcionar uma interação simples e intuitiva para o usuário, com design responsivo e funcionalidade eficiente.
